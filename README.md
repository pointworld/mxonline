# mxonline

## 项目启动

```text
git clone https://github.com/pointworld/mxonline.git
$ cd mxonline
$ pip install -r requirements.txt
$ python manage.py runserver
```

## 开发环境

- 操作系统:         MacOS@10.13.5
- 开发工具:         PyCharm@2018.03
- 后台语言:         Python@3.7.1
- 后台框架:         Django@2.1.4
- 数据库:           MySQL@5.7
- 数据库管理工具:    Navicat@12.1.13
- 版本管理工具：     Git
